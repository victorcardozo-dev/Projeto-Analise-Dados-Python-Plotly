# Projeto- Análise de dados com Python
 Projeto de análise de dados usando o framework Plotly para a criação de dashboards

## Apresentação do projeto

- Vamos fazer uma análise de uma base de dados para verificar essas informações dos clientes e verificar o percentual de cancelamento desses clientes.

- Como  é um projeto de Análise, o objetivo vai ser fazer análises para ver onde temos os maiores cancelamentos e os motivos, com isso vamos poder propor uma solução para diminuir quantidade de clientes cancelando.

## Como rodar o projeto

### Criar ambiente virtual
py -m venv .venv

### Ativar ambiente
.venv\Scripts\activate

### Instalar dependências
pip install -r requirements.txt

### Rodar aplicação
E é só executar o arquivo main.ipynb
